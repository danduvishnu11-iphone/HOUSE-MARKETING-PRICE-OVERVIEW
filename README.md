# 🏠 Housing Market Analysis Dashboard

## 📌 Project Overview

The **Housing Market Analysis Dashboard** is an interactive Tableau visualization project designed to analyze housing market trends, pricing patterns, and property characteristics.

This dashboard provides meaningful insights into:
- Housing price trends over time
- Relationship between property size and sale price
- Distribution of houses based on quality ratings

The project demonstrates data visualization and business intelligence skills using Tableau.

---

## 📊 Dashboard Features

### 📈 Price Trend Over Time
Visualizes how housing sale prices have changed across different construction years.

### 🏡 Price vs Area Analysis
Shows the relationship between ground living area (`Gr Liv Area`) and sale price using a scatter plot.

### ⭐ Houses by Quality
Displays the count of houses categorized by overall quality ratings.
